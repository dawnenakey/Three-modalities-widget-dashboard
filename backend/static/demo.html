About PIVOT

Our mission is to ensure that everyone, regardless of ability or language, can access information online.

How It Works

The PIVOT widget is embedded on your website using a simple script tag. It communicates with the PIVOT API to fetch accessibility content for each page. Content is automatically matched to the current URL, so visitors always see relevant accessibility options.

Website administrators use the PIVOT dashboard to:

  • Add websites and pages
  • Upload ASL videos and audio files
  • Generate AI-powered audio using text-to-speech
  • Configure widget appearance and behavior
  • Track usage analytics

The entire setup process takes less than 5 minutes, and content can be updated at any time through the dashboard.

Technical Details

Widget Installation:

Add this code to your website (replace YOUR_WEBSITE_ID with your actual ID):

<script src="https://translocate-2.preview.emergentagent.com/widget.js" 
        data-website-id="YOUR_WEBSITE_ID"></script>

Customization Options:

  • data-website-id - Your website ID (required)
  • data-position - Widget position: bottom-right, bottom-left, top-right, top-left
  • data-primary-color - Primary color (default: #1e3a8a)
  • data-accent-color - Accent color (default: #f97316)
